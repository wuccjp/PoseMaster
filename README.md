# 課程名稱：AI 肢體感應實戰課 —— 從視覺辨識到創意互動
## AI 視覺初體驗  
+ 電腦視覺原理：電腦如何「看見」人體？
++ 展示 MoveNet 模型：電腦如何從雜亂的影像中抽取出 17 個關節點？
+ 環境建置：p5.js Web Editor 與 ml5.js 1.0 引入。 
+ 實作：開啟視訊鏡頭，顯示第一個 AI 偵測畫面。
++ 任務： 找出「鼻子(0)」、「左手腕(9)」與「右手腕(10)」的編號。
++ 體驗： 觀察當手被遮住或轉身時，AI 的信心分數（Confidence）變化。
## 解析 17 個生命密碼 (Keypoints)
+ 深入了解 COCO 模型 17 個關鍵點位。
+ 實作：【17 點編號調試器】，親手標註自己的鼻子、手腕與肩膀。
+ 技術重點：陣列 (Array) 與 迴圈 (Loop) 的應用。
